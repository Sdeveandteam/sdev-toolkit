# sdev-toolkit
Automated Multi-Scanner Tool for Lab Project.

## Features
- Nmap Network Scanner
- Sqlmap SQLi Scanner
- SQL Database Dump
- Vulnerability & Bug Scanner

## How to Install & Run (Termux / Linux)
```bash
pkg install git nmap sqlmap mariadb nikto -y # Jika di Termux
git clone [https://github.com/Sdeveandteam/sdev-toolkit.git](https://github.com/Sdeveandteam/sdev-toolkit.git)
cd sdev-toolkit
chmod +x main.sh
./main.sh
